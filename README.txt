For the Derpnet IRC server project.

The folder docs contains the new set of RFCs for IRC (numbers 2810-2813). The old
RFC (number 1459) is superceded by these.  In addition, our own technical
documents are in this folder.

We have an IRC channel: #derpnet at irc.foonetic.net. Our faithful bot DerpBot
will tell us whenever someone pushes an update.
